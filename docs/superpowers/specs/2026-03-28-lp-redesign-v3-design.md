## 背景と目的

- Voice AI / Claude Codeの紹介が前面に出すぎてアプリが埋もれている
- SNSの活動が一目でわからない
- ダークテーマ（宇宙風）が医療系LPとして適切でない

### ゴール（優先順位順）
1. アプリを試してもらう（βテスト登録）
2. 他のSNSもフォローしてもらう
3. 「この人すごい」と思ってもらう（信頼・権威性）
4. お問い合わせ

### ターゲット
- Xのプロフィールリンクから来る医療従事者が主な流入元

- **Tivoli om-tivoli風** — 北欧のポップさと上品さの共存

- フルブリード写真を多用したビジュアル重視レイアウト

## ページ構成

### 1. Hero（フルブリード写真）
- コペンハーゲンの風景写真をフルブリードで表示
- テキスト重ね: 「🇩🇰 Copenhagen, Denmark」→ 「麻酔科医X」（大見出し）→ 一行紹介 → CTA
- Tivoli om-tivoliのHeroと同じ構成

### 2. Apps（横スクロール写真カード）
- Tivoli「Tæt på Tivoli」セクションと同じカルーセル形式
- 各アプリのスクショを背景にしたカード（グラデーションオーバーレイ + タイトル + 説明）
- 6アプリ: Senere, PreVisit, TOFun, Summary, AneTimer, ToughCall
- カルーセルナビ（‹ ›）付き

### 3. Feature Pick Up × 2（2カラム交互）
- Tivoli「Start din karriere」セクションと同じレイアウト
- **Senere**: テキスト左 + アプリスクショ右
- **Summary**: スクショ左 + テキスト右
- 各セクションにCTAリンク

### 4. SNS（ダークセクション）
- ダーク背景でコントラストを付ける
- 5カードグリッド: X, Instagram, YouTube, TikTok, note
- 各カードに発信内容 + 更新頻度

### 5. Profile（2カラム: 写真 + テキスト）
- プロフィール写真左 + テキスト右
- 名前、肩書、バイオ、タグ
- Voice AIはタグ内で軽く言及

### 6. CTA（フルブリード写真バナー）
- 写真背景 + ダークオーバーレイ
- 「βテスター募集中」+ CTA ボタン

### 7. Footer
- ダーク背景
- ブランド名 + リンク + コピーライト
